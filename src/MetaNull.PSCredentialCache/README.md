# MetaNull.PSCredentialCache

Windows-only DPAPI credential caching with configurable registry storage and expiry.

## Install

```powershell
Install-Module MetaNull.PSCredentialCache -Repository PSGallery
```

## Dependency

- MetaNull.PSStringToolkit

## Highlights

- DPAPI protect/unprotect helpers
- Cached credential create, read, test, and remove
- Configurable registry root and cache duration
