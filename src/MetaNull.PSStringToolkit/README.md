# MetaNull.PSStringToolkit

String, encoding, conversion, placeholder expansion, `.env` parsing, and filename validation utilities.

## Install

```powershell
Install-Module MetaNull.PSStringToolkit -Repository PSGallery
```

## Highlights

- HTML and URL encoding helpers
- Camel case and label conversion
- `.env` import/export helpers
- Placeholder expansion
- Filename validation
