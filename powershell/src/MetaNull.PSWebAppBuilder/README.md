# MetaNull.PSWebAppBuilder

Build and deployment pipelines for Laravel, Lumen, and Node.js applications.

## Install

```powershell
Install-Module MetaNull.PSWebAppBuilder -Repository PSGallery
```

## Dependency

- MetaNull.PSGitOps

## Highlights

- Generic deployment pipeline via `Invoke-WebAppUpdate`
- Laravel and Lumen API build wrappers
- Node.js client build and deploy wrapper
- Git-only update workflow
