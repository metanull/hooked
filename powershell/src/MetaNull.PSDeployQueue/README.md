# MetaNull.PSDeployQueue

Registry-backed deployment queue with Task Scheduler runner integration and notification helpers.

## Install

```powershell
Install-Module MetaNull.PSDeployQueue -Repository PSGallery
```

## Highlights

- Mutex-protected queue operations
- Task Scheduler queue runner install/start/query helpers
- Notification queue and SMTP dispatch
- Configurable registry and task scheduler settings
